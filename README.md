## Contenidos

Este es el repositorio de calculadora y proyecto final de Electronica Digital Unal 2026-1, este esta dividido en 2 carpetas principales las cuales son:

* Algoritmos: donde esta todo lo relacionado a la calculadora.
* Proyecto: donde esta todo relacionado al proyecto de la pantalla.

Para la calculadora, se encontraran primero todos los diagramas y la logica, dentro de esta estaran los core con los codigos Verilog que vienen con los Testbench de cada modulo y archivos de simulación gtkwave e modulos correspondientes a cada Algoritmo.
